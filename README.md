## 1.系统特性

### 多仪器联合控制

* 通过交换机局域网联通多种Visa设备，通过控制板电路改装联通多种串口设备。
* 已支持常见采样动作
* 支持快速脚本定制复杂采样动作

### 友好的人机交互

* v5.0前支持命令行交互式控制，如有改进意见请联系作者
* v6.0计划支持GUI上位机

### 自动化特性

* 自动设置Tx、Rx参数，自动开启信号
* 自动绘制采样数据曲线，自动展示
* 自动保存采样数据，支持txt、csv、xlsx三种数据保存格式
* 单次扫频和跨步旋转采样时间在分钟量级，连续旋转采样时间在秒量级
